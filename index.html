SmallColorPicker

SmallColorPicker is a library which allows you to create a minimalistic color picker with jQuery. Also it contains CSS styles for color selection buttons.

Color picker

Here's it:

Buttons

Buttons are in separate CSS file, color-buttons.css; this CSS has no dependencies and can be used standalone.

Use this snippet to create buttons: 

<div class="color-btn color-btn-left"><b style="background-color: #ff66bd;"></b>text</div>

Browsers suport

SmallColorPicker makes use of CSS3 features. Supported browsers are Chrome, Safari, Firefox, Opera, IE≥9.
