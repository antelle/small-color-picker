SmallColorPicker

SmallColorPicker is a library which allows you to create a minimalistic color picker with jQuery. Also it contains CSS styles for color selection buttons.

Installation

Download the library, include CSS and JS files and write some initialization code (refer to SmallColorPicker.defaults in the source code for available options).

expand example

var colorPicker = new SmallColorPicker.CirclePicker({
    placement: {
        position: "static",
        parent: $("#color-selector-box")
    },
    colors: {
        colorOld: "#990e8d",
        colorNew: "#6d1f7a"
    },
    behavior: {
        hideOnSelect: false
    },
    events: {
        ok: function(color) {
            showColorPickerText("Selected color: " + color);
        },
        cancel: function(color) {
            showColorPickerText("Cancelled");
        }
    }
});
colorPicker.show();

Browsers suport

SmallColorPicker makes use of CSS3 features. Supported browsers are Chrome, Safari, Firefox, Opera, IE≥9. Buttons can be used in all browsers.
